2. Welche Daten wir verarbeiten

Website (flomily.de)

Beim Besuch unserer Website werden automatisch technische Daten erhoben: IP-Adresse, Browser-Typ, Betriebssystem, Zugriffszeitpunkt. Diese Daten werden von unserem Hosting-Anbieter Cloudflare verarbeitet.

Warteliste / Anmeldeformular

Wenn du dich fuer die Beta anmeldest, erheben wir: Name und E-Mail-Adresse. Diese werden ueber den Formular-Dienst Formspree verarbeitet.

Telegram-Bot

Bei Nutzung des flomily Telegram-Bots verarbeiten wir:

  • Telegram-ID und Telegram-Benutzername
  • Nachrichteninhalte (Text, Fotos, Dokumente, Sprachnachrichten)
  • E-Mail-Adresse (bei Registrierung angegeben)

Termin-Versand (ICS)

Erkannte Termine werden als Kalendereinladung (ICS-Datei) per E-Mail verschickt. Dabei verarbeiten wir: Name und E-Mail-Adresse der Empfaenger sowie die Termindaten.

Drip-Mails

Registrierte Nutzer erhalten automatisierte Service-E-Mails (Willkommen, Tipps, Updates). Dabei speichern wir: E-Mail-Adresse, Name und welche Mails bereits verschickt wurden.

4. KI-Verarbeitung

Wichtiger Hinweis: Deine Nachrichten an den Telegram-Bot werden von einer kuenstlichen Intelligenz (Claude, Anbieter: Anthropic, USA) verarbeitet, um Termine zu erkennen. Die Verarbeitung erfolgt auf Servern in den USA.

Sprachnachrichten werden lokal auf unserem Server transkribiert (whisper.cpp) und nicht an externe Dienste gesendet. Nur der transkribierte Text wird zur Termin-Erkennung an die Claude API weitergeleitet.

Die Datenuebermittlung in die USA erfolgt auf Basis von Standard Contractual Clauses (SCCs) gemaess Art. 46 Abs. 2 lit. c DSGVO.

Du wirst bei der Aktivierung des Bots ausdruecklich auf die KI-Verarbeitung hingewiesen und musst dieser gesondert zustimmen.
